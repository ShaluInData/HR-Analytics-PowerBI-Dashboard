# HR-Analytics-PowerBI-Dashboard
HR Analytics Dashboard using Power BI
* Project Overview :

This project analyzes employee attrition trends using Power BI. The goal is to identify key factors influencing employee turnover and provide actionable insights for HR decision-making.

* Objectives :

Analyze employee attrition rate
Identify high-risk job role
Study attrition by age, salary slab, education, and years at company
Compare attrition across gender

* Tools & Technologies Used :
  
Power BI
DAX
Excel (CSV Dataset)
Data Cleaning & Transformation using Power Query

* Key KPIs :

Total Employees: 440
Total Attrition: 49
Attrition Rate: 11.1%
Average Age: 37
Average Salary: 6.8K
Average Years at Company: 7.2																																				
																																											* * Key Insights :

Highest attrition observed in Sales Executive role
Employees earning between 5k–10k show higher attrition
Age group 26–35 has maximum attrition
